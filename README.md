# Vista
A dedicated project where I build a practical tool for language immersion.

## 背景
本專案希望可以解決自身對於語言學習以及程式設計學習的慾望
因市面上的語言學習軟體，如:XXlingo，不太符合本身學習的需求，且訂閱費實在太高昂，故想要開發一個免費的平台
除了可以客製化所有的功能以外，還可以讓開發的能力變強

## 預計使用工具
Vue v3: 用以建置前端網頁的環境
FastAPI: 用以當作前後端的接口，這邊選擇FastAPI是因為他最輕量，對一個MVP的專案來說非常適合
Python: 作為後端主要的程式語言
Postgre: 若有什麼資料需要儲存，會使用postgre當作後端的資料庫
AWS S3、cloudfront、router53: 作為建置網站的根基，可以使用CI/CD來管理
