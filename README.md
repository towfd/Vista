# Vista
## 背景
我是一個喜歡學習語言的人，常常在思考著怎麼樣去學習會讓我效率最高，且可以維持我的熱情很久
嘗試過很多的方法，例如：
* 去買參考書來看
* 去考試讓自己有目標
* 使用市面上的軟體，如：Duolingo、drop等等
* 看原文的影片或漫畫，然後一個字一個字翻譯

一開始的話通常我會先使用Duolingo或是drop去熟悉一個語言的字母、聲調以及基本的發音
然後學習比較基本的單字、句型，往往前期的效果是非常的好的
但是到中後期反而會開始比較吃力，由於一些句子比較難在現實生活上面使用，很多時候會想，為什麼我要學這些句型，然後就慢慢地消磨學語言的熱情
再來我會去報名一些考試，因為考試對我來說是最有效率的學習方式，去考試的話會比較有目標且有時間限制，在壓力比較大的時候去學習，效率會比較好，但是非常的辛苦，每考完一次試感覺就要休息好一段時間才可以繼續去準備，若考不好的話反而會有反效果來讓自己的熱情消散
看漫畫或是影片固然是會讓自己覺得很放鬆，因為有時候看不懂的話可以用畫面來猜測現在大概是在說什麼，但是因為看漫畫通常都是用手機在看，每次看的時候還要再額外的滑出去用google翻譯，對我造成了極大的阻礙，有時候滑出去了就不想再繼續看了，進度很緩慢
